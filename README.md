# financial-chat

Chat using websockets

## Commands
- How to run:
    - To run you must have docker installed
    - `docker-compose up -d`
    - `make start` or `go run .`
    - Access `http://localhost:8081`
- How to stop:
    - `make stop`
- How to run tests:
    - `make test`
